# Pd-Ready Kawaii Future Bass Samples

These files were converted from the local `Clark Audio - Kawaii Drum Kit BETA` pack already present in `samples/kawaii_future_bass`.

The conversion normalizes selected samples to safe Pd filenames and 16-bit mono 44.1 kHz WAV so `[soundfiler]` can load them reliably from patch messages.
