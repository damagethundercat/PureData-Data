Frutiger Aero Pure Data Package

Open frutiger_aero.pd in Pure Data.

Keep these WAV files in the same folder as frutiger_aero.pd:
- cute-bloop.wav
- cute-water.wav
- cute-pop.wav
- cute-chirp.wav

The patch loads the samples with relative file names, so the folder can be
moved anywhere as long as these files stay together.
